# AurumAccounting

Modern, multi-language accounting web application for small businesses and freelancers.

## Features

- **Dashboard**: Summary cards (revenue, expenses, profit, accounts count), recent transactions widget
- **Chart of Accounts**:  Manage accounts (assets, liabilities, equity, revenue, expenses) with search and filter
- **Journal Entries**: Create double-entry transactions with debit/credit auto-balance validation
- **Trial Balance Report**: Period-based report with account balances
- **Multi-language**: i18n support (EN default, ready for AZ/RU/TR)
- **Responsive**:  Mobile-first design (360px to 1440px+)
- **Accessibility**:  Keyboard navigation, ARIA labels, focus management

## Tech Stack

- **Frontend**: Vanilla HTML5, CSS3 (BEM), JavaScript (ES6 modules)
- **No frameworks**: Pure JavaScript for maximum control and performance
- **Storage**: LocalStorage (mock data for frontend MVP)
- **i18n**: Custom implementation with JSON translation files
